Introdução às Tags HTML5

As tags HTML5 são os blocos de construção essenciais para criar páginas web modernas e interativas.

Tags Mais Utilizadas

Tag Descrição Função 1 Função 2
<!DOCTYPE> Define o tipo de documento HTML. Indica a versão do HTML: <!DOCTYPE html>
<html> Define o início e o fim do documento HTML. Contém todo o conteúdo da página.
<head> Contém metadados sobre o documento. Define configurações da página, como título.
<title> Define o título da página exibido na aba. Exibe um título na aba do navegador.
<meta> Fornece informações meta sobre o documento. Define a codificação de caracteres. Melhora a indexação de mecanismos de busca.
<link> Vincula um recurso externo, geralmente CSS. Liga a um arquivo de folha de estilo CSS.
<script> Insere scripts, geralmente JavaScript. Adiciona interatividade à página. Carrega recursos externos assincronamente.
<body> Contém o conteúdo visível da página. Todo o conteúdo exibido aos usuários.
<h1> até <h6> Define cabeçalhos de diferentes níveis. Organiza o conteúdo em seções. Indica hierarquia de informações.
<p> Define um parágrafo de texto. Agrupa o texto em parágrafos.
<a> Cria um hiperlink para outras páginas. Navega para outras páginas ou recursos. Pode abrir links em novas abas/janelas.
<img> Exibe uma imagem. Mostra imagens na página. Pode melhorar a experiência visual.
<ul>, <ol>, <li> Cria listas não ordenadas e ordenadas. Organiza informações em listas.
<div> Cria uma divisão ou contêiner genérico. Agrupa elementos para estilização ou lógica. Melhora a estruturação do layout.
<form> Cria um formulário para entrada de dados. Permite a interação do usuário. Envia dados para processamento.
<input> Cria um campo de entrada de dados. Coleta informações do usuário. Diversos tipos: texto, senha, etc.
<button> Cria um botão interativo. Ativa ações quando clicado. Pode enviar formulários.
<table>, <tr>, <td> Cria tabelas e suas células. Organiza dados em formato tabular. Apresenta dados de maneira organizada.
<iframe> Exibe uma janela incorporada para outro conteúdo. Incorpora conteúdo de outras fontes. Inclui mapas, vídeos, etc.
<audio>, <video> Incorpora arquivos de áudio e vídeo. Reproduz mídia diretamente na página. Oferece conteúdo multimídia.
<footer> Define o rodapé da página ou seção. Informações finais ou créditos.
